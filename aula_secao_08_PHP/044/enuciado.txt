No arquivo index.php temos um formulário html com dois inputs.
O script de tratamento.php está vazio.
O objetivo é escrever o código no ficheiro tratamento.php para alcançar o seguinte resultado.

1. Se o script for acessado diretamente, deve apresentar um erro.
2. Os inputs devem ser validados. Se algum dos valores não for um valor númerico positivo, volta para o formulário e apresenta esse erro.
3. sendo os dois inputs númericos, apresentar como resultado a multiplicação do primeiro pelo seundo e dar o exercício como terminado.
